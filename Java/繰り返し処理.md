# while文
```
while (条件) {
  繰り返す処理；
}
```
```
int n = 1;
while (n <= 5) {
  System.out.println(i + "回繰り返しました。");
  n++;
}
結果：
1回繰り返しました。 // n <= 5がtrue
2回繰り返しました。 // n <= 5がtrue
3回繰り返しました。 // n <= 5がtrue
4回繰り返しました。 // n <= 5がtrue
5回繰り返しました。 // n <= 5がtrue
               // n <= 5がfalseのため、6回繰り返しましたと出力されない。
```
